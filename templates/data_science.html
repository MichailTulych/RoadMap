Рекомендуемые Пет-Проекты

Anti-Fraud Simulator

Цель: Веб-демо, симулирующее поток транзакций с обнаружением мошеннических.

Роль Data Scientist:

  • Найти датасет мошеннических транзакций (Kaggle)
  • Провести EDA и генерацию фич
  • Обучить модель LightGBM/CatBoost
  • Создать FastAPI микросервис для инференса
  • Обернуть решение в Docker-контейнер

Стек: Python, Pandas, LightGBM/CatBoost, FastAPI, Docker

RAG-powered Q&A Assistant

Цель: Частный помощник с знаниями из ваших документов.

Роль Data Scientist:

  • Разбить документы на чанки
  • Создать эмбеддинги с Sentence Transformers
  • Настроить векторную базу (ChromaDB/FAISS)
  • Реализовать логику поиска и генерации ответов
  • Создать FastAPI сервис

Стек: Python, Sentence-Transformers, ChromaDB/FAISS, OpenAI API, FastAPI

Recommendation System API

Цель: Микросервис для рекомендации похожих товаров.

Роль Data Scientist:

  • Реализовать Content-Based фильтрацию (TF-IDF/BERT)
  • Реализовать Collaborative Filtering (SVD)
  • Сравнить результаты подходов
  • Создать FastAPI сервис для рекомендаций

Стек: Python, Pandas, Scikit-learn, Sentence-Transformers, FastAPI
